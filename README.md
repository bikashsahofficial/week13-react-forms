# Week 13 React Forms Complete

This project includes the required React Forms assignment and the optional Context and Upload assignments.

## Included

- Forms assignment
  - LoginForm
  - RegisterForm
  - Logout view
  - useForm custom hook
  - login/logout routing

- Optional Context assignment
  - UserContext
  - contextHooks
  - ProtectedRoute
  - user state shared across components

- Optional Upload assignment
  - Upload view
  - file input preview
  - FormData upload function
  - media post function
  - upload route

## How to run

```bash
npm install
npm run dev
```

Open the URL shown in terminal.

## How to test build

```bash
npm run build
```

## Important note

The real Metropolia APIs require a valid user account and token. The UI works without login, but real upload and profile functions need a successful login token.
